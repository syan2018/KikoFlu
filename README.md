## 特性

### 🎵 强大的音频播放
- **智能缓存系统** - 自动缓存已播放内容，离线也能流畅播放
- **后台播放** - 支持系统通知栏控制，锁屏播放无压力
- **多种播放模式** - 单曲循环、列表循环、随机播放
- **播放速度调节** - 0.5x - 2.0x 倍速播放
- **歌词显示** - 自动加载并同步显示歌词

### 🎨 精美的界面设计
- **Material Design 3** - 遵循最新设计规范
- **动态主题** - 支持从系统提取颜色（Android 12+）
- **深色模式** - 完美适配明暗主题切换
- **流畅动画** - 优化的页面转场和交互反馈

### 👥 多账户支持
- **账户管理** - 可添加和切换多个 Kikoeru 账户
- **快速切换** - 一键切换不同账户，无需重复登录
- **数据隔离** - 每个账户独立管理收藏和历史记录

### 🔍 高效搜索浏览
- **智能搜索** - 支持按标题、社团、声优、标签搜索
- **高级筛选** - 多维度筛选作品（标签、评分、发售日期等）
- **作品详情** - 完整展示作品信息、评论、文件列表
- **图片浏览** - 支持作品封面和图集查看

### ⚙️ 灵活的设置选项
- **服务器管理** - 支持自定义服务器地址，可测试连接延迟
- **缓存控制** - 自定义缓存大小限制和清理策略
- **界面定制** - 主题模式、配色方案自由选择
- **播放设置** - 音质选择、倍速播放等个性化配置

### 📥 离线下载管理
- **批量下载** - 支持整个作品或选择性下载
- **断点续传** - 下载中断后可继续，不浪费流量
- **下载队列** - 智能管理下载任务，支持暂停/恢复/取消
- **存储管理** - 自动清理过期缓存，可设置缓存大小限制

## 开始使用

- Flutter SDK 3.0 或更高版本
- Dart SDK 3.0 或更高版本
- Android Studio / VS Code

```bash
git clone https://github.com/Meteor-Sage/Kikoeru-Flutter.git
cd Kikoeru-Flutter
```

```bash
flutter pub get
```

```bash
flutter build apk --release
```

```bash
flutter build ios --release
```
